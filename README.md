# Mallorca · Guía 6 Días

Guía slow travel PWA: **playas, paisajes y pueblos** por toda la isla.

- **Web:** https://520520u.github.io/MALLORCA/
- **Short link:** https://520vip.space/MALLORCA

## Regenerar

```bash
cd mallorca-guia
python3 build.py
python3 setup-pwa.py
cp -R ../mallorca-pwa/* ../docs/
```

## GitHub Pages

Branch `main`, carpeta `/docs`.

## Imágenes extras

```bash
python3 download_images.py   # Wikimedia → images/
```
